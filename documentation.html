Validation

At times you'll need to display input validation information to a user. While a modal is sometimes an acceptable solution, it is jolting and can feel overwhelming. Validation classes are the perfect way to alert a user that the information provided was insufficient. With Exprecss' beautifully simple approach you are now able to alert a user without being too distracting. The validation text is fired only when the client side validation comes back as insufficient.

<div class="row">
    <div class="col-4"/>
        <label for="">Validation Example</label>
        <span class="validation-text validation-text-important">Oops! You forgot to enter a valid email.</span>
        <input type="text" class="input-default" placeholder="Email">
        <a href="" class="btn btn-action btn-validate-me">Validate Me</a>
    </div>
</div>
Oops! You forgot to enter a valid email. Validate Me
