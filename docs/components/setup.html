Initial Setup

Setting up Exprecss is simple. First include exprecss.css in your website's <head>.

<head>
    <link rel="stylesheet" href="css/exprecss.css">
</head>

Lastly you'll need to include exprecss.js just before the closing </body> tag.

<body>
    .
    ..
    ...
    <script scr="js/exprecss.js">
</body>

If there are no 403 errors related to Exprecss then Exprecss has been setup correctly!
